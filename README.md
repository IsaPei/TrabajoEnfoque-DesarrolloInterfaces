Proyecto para mostrar los resultados de una base de datos a través de una interfaz que a su vez generará un informe (documento pdf).

El proyecto contiene una base de datos creada en local, PHPmyadmin. 
Para poder abrirlo es necesario el uso de XAMPP (activar Apache y SQL) e importar la base de datos en PHPmyadmin.
Usuario: root
Contraseña: (dejar vacío)
Para que se puedan mostrar los resultados correctamente y que se genere el informe,
los datos introducidos en los campos de la interfaz deben coincidir entre ellos.
